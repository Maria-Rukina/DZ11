# Домашнее к работе № 11

Условие задачи

С одномерным массивом, состоящим из n вводимых с клавиатуры
положительных и отрицательных целых чисел выполнить следующие
вычисления: Отношение количества четных значений элементов массива к числу
положительных.

1.Алгоритм и блок-схема

Алгоритм

1. Начало

2. Ввести n чисел массива через scanf

3. Вводим числа массива через цикл for перебором i до конца n

4. Через цикл for перебираем цисла массива и с помощью if проверяем их и записываем их кол-во к переменным s(чётные числа) и p(положительные числа)

5. Считаем соотношение чётных чисел к числу положительных, но используем double для большей точности: double o = (double) s / p

6. Ввыводим соотношение o

7. Конец
